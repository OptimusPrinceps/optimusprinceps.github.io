# Run locally

Install Ruby:

```bash
brew install ruby
echo 'export PATH="/opt/homebrew/opt/ruby/bin:$PATH"' >> ~/.zshrc
```

then:

```bash
bundle install
bundle exec jekyll serve --livereload
```

View at http://127.0.0.1:4000
